# Это что?
Это есть веселый школьный проект в рамках задания от ТБанка. Нашему классу преподавали фронтенд (React группе, которая это делает), и теперь просят показать, что из этого мы запомнили.
Мы группой из 4 (количество людей было согласовано) человек придумали шуточную идею, которая будет использовать много различных элементов, что сможет оправдать количество людей в команде.
